⏰ Daily Reminder — Mon–Fri at 7:00 PM

Set a recurring alarm or calendar event labeled Daily Shipped Log at 7:00 PM, Mon–Fri.

iPhone: Settings → Focus → or just set a repeating Clock alarm. Label it "Daily Shipped."

Google Calendar: Create a recurring Mon–Fri event at 7 PM with a notification. Add your app URL to the event description.

📅 Weekly Review — Sunday at 8:00 AM

Set a Sunday 8 AM reminder. Open the app → Reviews tab → tap "Generate This Week's Mentor Review."

📈 Monthly Review

On the last day of each month → Reviews tab → "Generate This Month's Review."

📱 Add to Home Screen

iPhone/iPad (Safari): Tap the Share button → "Add to Home Screen" → tap Add.

Android (Chrome): Tap ⋮ menu → "Add to Home Screen."

Once added, it opens like a native app — full screen, no browser bar.
